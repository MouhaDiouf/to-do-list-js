# To Do List App 


## Technologies

- HTML
- CSS
- JavaScript



## Functionalities

Can be found on: ( <a href="https://www.theodinproject.com/courses/javascript/lessons/library"> The Odin Library Project </a>)


The user can:

- Create New Tasks
- Create New Projects
- Remove Taks and Projects
- Show All Taks and Projects
- Show Indidual Taks Details
